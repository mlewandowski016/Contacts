1. Aplikacja WEB'owa - lista kontaktów
2. Aplikacja pozwala na przechowywanie w bazie danych danych kontaktowych.
3. Aby aplikacja działała poprawnie należy połączyć się z bazą danych.
Jeśli nie mamy do niej dostępu możemy ją wygenerować za pomocą migracji oraz seedera.
W tym celu należy włączyć w Visual Studio konsolę menedżera pakietów (package manager console) i wpisać następujące linie:
- enable-migrations
- update-database
4. Aby móc dodawać, edytować oraz usuwać rekordy należy najpierw założyć konto, a następnie się zalogować (ważne aby po rejstracji potwierdzić konto).
